== Description ==

SendMsgs provides Push Notifications service for Web and Mobile. Grow your Business in Minutes. First Month Completely FREE!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You will find 'SendMsgs Settings' menu in your WordPress admin panel.
4. Select the right choice to show the plugin either in header or footer.
5. Enter URL's you want to add in the frontend website. If the jQuery is already included then please leave the first url field blank.
6. Select Page Options to include or exclude the script in pages, categories etc.
7. Save the settings and that's done. 

Thank you.

== Frequently Asked Questions ==

= Can I switch off the plugin for some days? =

Yes, you can. You can put the plugin ON or Off through the settings, no need to disable the plugin.

= Do I have to include jQuery? =

The jQuery is already initiated by the Wordpress or the Theme you are using. But yet if you want to include different version of the jQuery you can include it.

= Include or exclude specific pages = 
You can find an option in settings "Page Options". Here you get options to include or exclude the script from specific pages by selecting the option and pages as per your need.
